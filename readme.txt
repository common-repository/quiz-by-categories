== Description ==
This plugin enable you to display quiz by category. Each category contain many questions. The system remember the user score and display it after each question suite is completed.
The administrator can propose up to 5 answers per question and give a customized answer depending if the user give a good or bad answer.

== Installation ==
Add the following shortcode to your page : `[singlequiz]`.
To add and modify quiz and answer you must have either admin or editor role.
The interface will appear differently if you have modification right.
To test what the user will see you can try using another browser or a incognito tab.
